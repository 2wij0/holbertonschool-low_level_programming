# C - Recursion

## Description
This project covers the concept of recursion in C programming. It includes several functions that demonstrate how a function can call itself to solve problems by breaking them down into smaller sub-problems.

## Files
All the following files are scripts and programs written in C:

| Filename | Description |
| -------- | ----------- |
| `0-puts_recursion.c` | Prints a string followed by a new line. |
| `main.h` | Header file containing all function prototypes used in this project. |
| `_putchar.c` | Function that writes a character to stdout. |

## Requirements
* Compiled on Ubuntu 20.04 LTS
* Compiled with `gcc -Wall -Werror -Wextra -pedantic -std=gnu89`
* Code uses the `Betty` style.

## Usage
To compile the files, use:
`gcc -Wall -pedantic -Werror -Wextra -std=gnu89 _putchar.c 0-main.c 0-puts_recursion.c -o 0-puts_recursion`
